FUVARVELÜNK – frissített csomag

A csomagban átírva lett:
- minden domain: fuvarvelunk.hu
- márkanév: Fuvarozz Velünk
- CNAME / robots / sitemap
- fizetési logika: Utalás a sofőrnek / Készpénz a sofőrnek
- sofőr bankszámla mező a fuvar feladásánál

TEENDŐ:
1. Csomagold ki a ZIP-et.
2. A teljes tartalmát töltsd fel a GitHub repó gyökerébe, a régieket felülírva.
3. GitHub Pages-ben a custom domain legyen: fuvarvelunk.hu
4. Rackhost DNS zónában is a fuvarvelunk.hu legyen GitHub Pages-re állítva.
