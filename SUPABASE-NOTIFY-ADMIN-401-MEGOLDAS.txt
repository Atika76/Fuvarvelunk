A 401 hiba oka nem a Node.js volt, hanem az, hogy a notify-admin Edge Function újradeploy után JWT ellenőrzést kezdett kérni.

A weboldalban jelenleg sb_publishable_ kulcs van a js/config.js fájlban, ez nem ugyanaz mint a régi anon JWT token, ezért a function 401 Unauthorized hibát adott.

Javítás ebben a ZIP-ben:
- visszaállítva a működő webes kód
- bekerült: supabase/config.toml
- a notify-admin és cleanup-expired-trips functionnél verify_jwt = false

TEENDŐ:
1. A ZIP fájljait írd felül a projektben / GitHubon.
2. PowerShellben ebben a projektmappában futtasd:
   npx supabase link --project-ref qkppqjcazakocxgxtlzc
   npx supabase functions deploy notify-admin
3. Ha akarod a lejárt fuvar takarítót is újratelepíteni:
   npx supabase functions deploy cleanup-expired-trips

Ha azonnal, config.toml nélkül akarod deployolni:
   npx supabase functions deploy notify-admin --no-verify-jwt

Ezután a notify-admin hívás újra menni fog, és megszűnik a 401 hiba.
