# FuvarPortál – GitHubra feltölthető induló projekt

Ez egy Oszkár-szerű fuvarmegosztó weboldal **statikus induló alapja**.

## Mit tud most?
- főoldal gyors keresővel
- fuvarlista és szűrés
- fuvar feladása űrlap
- admin mintafelület
- kitölthető alapadatok
- minden személyes adat helyett minta / üres mezők

## Fontos
Ez a verzió **GitHub Pages-re feltölthető**, de az adatok csak a böngésző helyi tárolójában (localStorage) mentődnek.

Ez azt jelenti, hogy:
- demóhoz jó
- kinézet és működési logika teszteléséhez jó
- valódi többfelhasználós rendszerhez később **Supabase** kell

## Feltöltés GitHubra
1. Hozz létre új repót.
2. Másold be a ZIP kicsomagolt fájljait.
3. A főoldal fájl neve `index.html`.
4. Kapcsold be a GitHub Pages szolgáltatást.

## Következő fejlesztési lépés
A következő körben ezt kell megcsinálni:
- Supabase projekt bekötése
- valódi adatbázis
- valódi admin jóváhagyás
- közös adatok minden eszközön
- később fizetős kiemelés
